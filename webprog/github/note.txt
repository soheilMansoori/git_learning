how to clone one repo from github or gitlab ? 
    git clone <repo_url>

how to show remotes ?    
    git remote

how to add remote ? 
    git remote add <remote_name> <remote_url>    


how to push repo ?
    git push <remote_name> <branch_name>    


how to show all remote branches ? 
    git branch -r    

how to switch to remote branch ? 
git first check local branch after that try to get branch from github
command :
    git switch <branch_name>     


how to fetch repo ? 
    git fetch <remote_name> <branch_name> 


how to pull repo ? 
    git pull <remote_name> <branch_name>     

how to create remote branch an push ?     
    git push -u <remote_name> <branch_name>

how to show remotes url ? 
    git remote -v
